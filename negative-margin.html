Le but de cette démo est de comprendre le fonctionnement de negative margin dans un but de layout full-height.

Dans les trois exemples on assigne la hauteur du viewport égale à la hauteur de la page. Sauf que :

  • dans le deuxième exemple on ajoute 
            .view { overflow: hidden; } 
        .viewport { margin-bottom: -40px; }
  • dans le troisième exemple on change la hauteur 
            .viewport { height: 260px; }

Même si 2 et 3 donnent le même résultat, le 2 peut être considéré comme meilleurs car le 3 suppose que le javascript connaisse la hauteur des menu header et footer et les retranche à la hauteur de page pour obtenir le résultat désiré.
